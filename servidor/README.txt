En el directorio server use el comando make en la terminal
para compilar el codigo.

Para iniciar el servidor y comenzar a escuchar conexion se debe ejecutar asi:

./server <puerto>

Por ejemplo, para ejecutar el servidor en el puerto 8080:

./server 8080
