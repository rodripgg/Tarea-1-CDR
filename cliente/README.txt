En el directorio cliente use el comando make en la terminal
para compilar el codigo.

Para iniciar el juego se debe ejecutar en terminal:

./cliente <ip> <puerto>

Por ejemplo, para conectar con un servidor en localhost en el puerto 8080:

./cliente 127.0.0.1 8080
